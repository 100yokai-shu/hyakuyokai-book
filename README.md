# 100-YOKAI Sales Ledger

100-YOKAI向けの宴/オンライン/その他売り上げ管理アプリです。

## 使い方

1. `index.html` をブラウザで開く
2. `商品管理` で宴、オンライン、その他の商品を登録する
3. `売上入力` で宴、オンライン、その他の売上を入力する
4. `月間成績` と `詳細集計` で売上、チェキ券枚数、動員、新規、前月比を見る
5. 必要に応じて `設定` で全体バック単価とカラーを変更する
6. `データ` からJSONバックアップを保存する

## 集計について

- 金額は `商品金額 x 数量` で計算されます。
- チェキ券枚数は `チェキ券換算 x 数量` で計算されます。
- バック総額は、全特典をチェキ券枚数に換算した合計枚数に応じて段階単価で計算されます。
- 段階単価は `0:405,50:425,100:455,150:485,200:520` のように入力します。
- 宴の `動員数`、`チェキ枚数`、`新規動員数`、`新規写メ枚数` は商品名に含まれる語句から分類します。
  例: `動員`、`チェキ`、`新規動員`、`新規写メ`

## データ保存

データはブラウザの `localStorage` に保存されます。
別端末や別ブラウザで使う場合は、`JSONバックアップ` と `JSON取込` を使ってください。

## スマホ利用

このアプリはPWA対応しています。
スマホのブラウザで開き、共有メニューから `ホーム画面に追加` を選ぶとアプリのように起動できます。

## Supabase同期

PCとスマホで同じデータを使うには、Supabase同期を設定します。

1. Supabaseで新規プロジェクトを作る
2. SQL Editorで `supabase.sql` の内容を実行する
3. Project Settings > API から `Project URL` と `anon public key` をコピーする
4. アプリの `データ` タブで `Supabase URL`、`anon key`、`自分専用同期ID` を入力する
5. `クラウドへ保存` を押す
6. 別端末では同じ設定を入れて `クラウドから読み込み` を押す

`自動同期` をONにすると、保存後と定期的なタイミングでクラウドへ保存します。
他メンバーとは別の `自分専用同期ID` を使ってください。同じIDを使うと同じデータとして同期されます。
